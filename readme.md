# Timeline For PrimeTween

Modified from this package [dotween-timeline](https://github.com/medvejut/dotween-timeline)

![Pic](/pic1.gif)

![Pic2](/pic2.gif)

## Warning

- need [Odin](https://odininspector.com/) to draw inspector
- Because of the limitations of PrimeTween, can't set cycles to -1.

## Easy To Extension

> You can inherit from ITimelineTrack to implement your own timeline track.

## How to use

- install [Odin](https://odininspector.com/) and PrimeTween [AssetStore](assetstore.unity.com/packages/slug/252960) [Github](https://github.com/KyryloKuzyk/PrimeTween)
- import this package
- add Timeline compoent on GameObject
- add tween track
